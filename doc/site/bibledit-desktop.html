1. Features

Bibledit-Desktop is a full-featured and mature desktop application. Features with:

  1. Native USFM text editor, several can be opened at once

  2. Project notes editor

  3. Chapter and book outline view

  4. Resizeable and moveable windows

  5. Easy application of styles

  6. Styles editor

  7. Selection and display of project notes by various parameters

  8. Online help

  9. Backup and restore

  10. Import and export

  11. Resources display

  12. Spelling check

  13. Simple or more advanced interface

  14. Merge projects.

  15. Collaboration through USB drive, or through the Internet

  16. Local or remote web application

  17. Interfaces to external programs including Xiphos and Bibletime.

And so on, and so on.

Bibledit was designed for a purpose: To give honour to God, and that people get access to the Word of God, and get saved through Jesus Christ.

You are invited to contribute to these pages.

Bibledit-Desktop is built with Gtk. It is a full-featured and mature desktop application.
