Bibledit-Gt 4.7 on Linux Mint 14.1

The instructions assume a standard installation of Linux Mint.

Connect to the installation media.

Start the Package Manager, and install the following list of packages, with their additional required changes:

build-essential libgtk2.0-dev git rcs curl sqlite3 libsqlite3-dev libxml2-dev libenchant-dev libgtksourceview2.0-dev libwebkit-dev texlive-xetex libgtkhtml3.14-dev

When through, open a Terminal.

Install bibledit.

Run bibledit.
