Bibledit-Gtk 4.6 on openSUSE 12.2

Bibledit-Gtk installation on a standard openSUSE installation with KDE desktop.

Connect to the Internet to be able to download software.

Install the following software packages: 

  • gtk2-devel 
  • sqlite3-devel 
  • gcc-c++ 
  • libxml2-devel 
  • git-core 
  • enchant-devel 
  • rcs 
  • gtksourceview2-devel 
  • libwebkitgtk-devel 
  • dbus-1-glib-devel 
  • make 
  • patch
Open http://software.opensuse.org in Firefox. Search for libgtkhtml. Choose libgtkhtml-3. Show the unstable packages for openSUSE 12.2. Do the 1-click install. Follow the screens, and complete it the installation procedure.In Suse Software Management, install package gtkhtml2-devel version 3.32.

Install bibledit.

Run bibledit.
