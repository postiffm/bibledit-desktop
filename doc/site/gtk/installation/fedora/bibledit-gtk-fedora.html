Bibledit 4.7 on Fedora 18

The instructions assume a standard installation of Fedora.

Connect to the installation media.

Start "Software".

Install the following list of packages:

gcc-c++ gtk2-devel sqlite-devel libxml2-devel git enchant-devel gtkhtml3-devel rcs patch gtksourceview2-devel webkitgtk-devel dbus-glib-devel libsoup-devel

(Installation can also be done from a terminal. To install gcc, for example, enter this: su -c 'yum install gcc')

Start "Terminal"

Install bibledit.

Run bibledit.
