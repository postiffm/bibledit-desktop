File Menu

Print book portion dialog

This dialog is accessed through menu File ▸ Print ▸ Project ▸ Change.

It allows you to select which portion of the project is going to be printed.

To print only a portion of a certain book, double click on the portion at the right of that book. Then you can choose to print from a chapter.verse till another chapter.verse.

To exclude a book from printing, just unselect it.

Everything can be done with either the mouse or with keyboard shortcuts.

Print project

This dialog is for printing a project.

It can be accessed through menu File ▸ Print.

Basic operation

Press button OK to let Bibledit generate a .pdf file with the project data.

Options

It is possible to print only a part of the project. The dialog shows whether the full project will be printed or only part of it. It does that at the "Portion:" label. Press button "Change" to change which portion will be printed.

Press "Options" to show more options.

Option "Write the references in the notes in full". This option scans the text to be printed in the footnotes, the endnotes, and the crossreferences. If it finds any references to verses of the Bible, it will include these with the text. For example if there is a crossreference that says "Acts 10.5; 1 Joh. 1:5", then it will include the full text of these verses in that crossreference. This can be useful for checking purposes, whether the crossreferences are correct.

Print Selecting books

In this dialog you can select which books you wish to operate on, either all books, or only the New Testament, or only the current book, and so forth. Individual books can be added or removed by clicking on them.

To select one or more books choose button "No books" and then click a book and the book gets selected.

If you want to choose another book, press Ctrl and click on the book.

Shift combined with click selects, if one book is selected already, all the other books between the selected and the clicked one.

There are hotkeys for No books, Old Testament, New Testament, All books, and Current book.

The Up/Down arrows allow to select a book, and Shift+Up/Down arrows select a sequence of books.

Ctrl+Up/Down goes to another book, and then press the space bar to select that particular book.

For example:

Move the cursor to the book you want to choose.

Use the spacebar to select the book.

Use Ctrl (keep it pressed) and move with the arrow-keys to another book.

Use the spacebar to select this book.

Go to the next book, and when the book is to be selected, press the space bar, and so on.
