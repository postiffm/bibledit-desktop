File Menu ▸ Project

Changes

The assistant for viewing changes in a project is accessible through menu File ▸ Project ▸ Changes.

It can show you the changes that have been made in the project.

1. You can view the changes since your last review.

2. You can view the changes made a date and time to be chosen by you.

3. You can view the changes that have been made between two dates and times to be chosen by you.

After pressing Apply, some processing will be done and then the changes will be presented to you.

Note. If a change is made in the text, and the changes are requested straight after that, the change may not have been recorded yet. In that case wait a short while and do it again.

Project Properties

In this dialog you can set the properties of a project.

The dialog is accessible through menu File ▸ Project ▸ Properties.

You can change the name of the project.

By default a project is editable, but you can set it non-editable. In that case the project cannot be changed or deleted.

You can add or remove books.

When adding books, you can either import a book from file, or add a book using an empty template.

You can set the versification system of the project, and the language. If the language is set the names of the books of the Bible can assume their localized names, instead of the English names. Some of the languages may have a localization of the numerals also. If this localization is there, the numbers of the chapters and verses in the printout change, for example the 1 may become a Û±.

You can set that a project depends upon another one, and that the dependency runs through a filter. This means that the text of the original project is taken, runs through a filter that changes the text, and then comes in the current project in the new form. This can be useful for creating a diglot. The filters can be managed in menu Preferences ▸ Filters.

There is also a setting for whether to check the spelling in the projects. If this option is ticked, then one or more dictionaries should be selected through the "Dictionaries" button. For dictionaries, see the dictionaries page.

Send / receive

This sends and receives your Bibles. It works when you have enabled collaboration for one or more Bibles. Normally the collaboration system send and receives the changes made in Bibles. It does that at intervals as set up by the user. When choosing Send / receive through the menu, sending and receiving Bibles is done immediately, regardless of the interval that was specified by the user.
